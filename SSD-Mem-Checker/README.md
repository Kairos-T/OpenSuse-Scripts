# SSD and Memory Health Test
Commands to identify PC issues by testing SSD and memory health.

Note: These were tested on [Ubuntu 22.04.4 LTS](https://ubuntu.com/download/desktop) live USB. 

## Warning

1. Perform these tests at your own risk!
2. Do not run the script directly, read and understand the script, and copy-paste the commands one by one.
3. This script works as of March 2024, and may not work in the future :(
